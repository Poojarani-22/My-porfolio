Parkinson’s Diseases Detection Project Developed a machine learning model using vocal attributes to detect Parkinson’s Disease. Achieved 92% accuracy with SVM by leveraging data from the UCI Machine Learning Repository. Utilized Python, Streamlit, and Pandas for data preprocessing, feature extraction, and model evaluation. Demonstrated potential for non-invasive early diagnosis.
